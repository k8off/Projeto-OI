Engines de Jogos

  • Unity
  • Game Maker
  • Unreal

Unity

Unity é uma das engines mais populares para criação de jogos 2D e 3D. É usada em jogos como Hollow Knight, Cuphead, e Monument Valley.

Game Maker

Game Maker é ideal para jogos 2D. É conhecida por jogos como Undertale e Hyper Light Drifter. Tem uma linguagem própria chamada GML.

Unreal

Unreal Engine é uma poderosa engine focada em gráficos realistas. É usada em jogos como Fortnite, Gears of War e muitos outros AAA.
